# nodejs-hw

Невеликий навчальний Node.js + Express проєкт із простими API-ендпоїнтами.

## Вимоги

- Node.js (рекомендовано 18+)
- npm

## Встановлення

```bash
npm install
```

## Змінні оточення

Створіть файл `.env` у корені проєкту:

```env
PORT=3000
NODE_ENV=development
```

## Запуск у режимі розробки

```bash
npm run dev
```

Сервер запускається через `nodemon`.

## Доступні маршрути

- `GET /` — повертає тестове повідомлення
- `GET /notes` — повертає список нотаток (заглушка)
- `GET /notes/:notesId` — повертає нотатку за ID (заглушка)
